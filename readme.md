# Arcgis 4.14 JSAPI

## Project setup
```
npm install
```

### Compiles typescript Files ( export ES6 module ) 
```
tsc -w
```

### Rollup for browser
```
npm run build
```

### Test
* should compile typescript files `tsc -w` ; before start unit test 
```
npm run test
```
